# HR Admin Project

This application consists of 3 parts, A SQL Server Database, a .NET 7 Web API and a React frontend using Vite

## Database
The SQL file used to build the database is included. 

`HR-Admin-Build.sql`


## Frontend
To setup the frontend, run 

```ps
cd Hr-Frontend
npm install
```

Once done, run 
```ps
npm run dev
```
